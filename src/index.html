Cara menginstal ReySkripsi Add-in di Word Desktop dan Web

Artikel • 10/06/2026 • 1 kontributor

ReySkripsi adalah Microsoft Word Add-in khusus yang dirancang untuk mempermudah dan mempercepat proses penulisan skripsi, tesis, maupun karya ilmiah lainnya. Dengan fitur pemformatan otomatis dan pengelolaan dokumen yang terintegrasi, ReySkripsi membantu mahasiswa fokus pada isi tulisan tanpa harus pusing memikirkan format margin, tata letak, atau standar penulisan kampus.

Langkah Instalasi (Sideload)

Artikel ini menjelaskan cara memuat (sideload) add-in ReySkripsi menggunakan file manifest.xml ke dalam Microsoft Word Anda. Metode ini berguna karena saat ini ReySkripsi dibagikan secara langsung tanpa melalui Windows AppSource.

Cara Instalasi Offline

Perhatian: Untuk menginstal add-in di Word Desktop, Anda tidak dapat menggunakan URL HTTPS langsung. Anda harus menggunakan Folder Jaringan Bersama (Katalog Tepercaya).

Sebelum melanjutkan, cukup unduh satu file installer di bawah ini. File ini sudah termasuk manifest Add-in di dalamnya sehingga Anda tidak perlu mengunduh file apa pun lagi secara terpisah.

Unduh reyskripsi-manager.exe

Ikuti langkah-langkah berikut untuk menginstal Add-in ReySkripsi di Word Desktop (Windows) dengan cepat tanpa perlu melakukan konfigurasi Trust Center secara manual:

  1. Unduh aplikasi reyskripsi-manager.exe dari tombol di atas. Anda bisa menyimpannya di folder mana saja.
  2. Klik Ganda (Double Click) aplikasi reyskripsi-manager.exe tersebut.
  3. Jendela Installer akan muncul. Klik tombol Install Add-in dan tunggu notifikasi sukses.
  4. Buka aplikasi Microsoft Word, lalu buat dokumen kosong (Blank Document).
  5. Pergi ke tab Home (atau Insert) > klik Add-ins > lalu cari menu Developer Add-ins (atau My Add-ins).
  6. Klik ReySkripsi untuk membukanya di panel samping (Taskpane).
Cara Uninstall: Buka kembali aplikasi Install-ReySkripsi.exe lalu klik tombol Uninstall. Add-in akan otomatis terhapus bersih dari sistem Anda.

Jika Anda lebih suka menggunakan versi browser web, prosesnya jauh lebih sederhana:

  1. Buka Word di Web dan buat dokumen kosong baru.
  2. Pergi ke tab Home atau Insert pada pita menu.
  3. Klik tombol Add-ins.
  4. Pada dialog Office Add-ins, pilih Upload My Add-in (Unggah Add-in Saya).
  5. Pilih file manifest.xml yang baru saja Anda unduh dan klik Upload.
  6. Add-in akan muncul di tab Home atau sebagai panel tugas di sebelah kanan.
