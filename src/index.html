Cara menginstal ReySkripsi Add-in di Word Desktop dan Web

Artikel • 10/06/2026 • 1 kontributor

ReySkripsi adalah Microsoft Word Add-in yang dirancang untuk mempermudah penulisan skripsi, tesis, maupun karya ilmiah. Fitur utama alat ini adalah kemampuannya untuk mendeteksi dan memiringkan (italicize) kata-kata asing secara otomatis berdasarkan Kamus Besar Bahasa Indonesia (KBBI). Dengan ReySkripsi, Anda tidak perlu lagi menghabiskan waktu mencari dan memiringkan kata bahasa asing satu per satu secara manual.

Fitur & Tampilan Aplikasi

Unduh & Instal Aplikasi

Untuk mulai menggunakan ReySkripsi di komputer Anda, silakan unduh aplikasi ReySkripsi Manager melalui tombol di bawah ini. Aplikasi ini dirancang agar proses pemasangan menjadi sangat mudah cukup dengan beberapa klik.

Unduh reyskripsi-manager.exe

Ikuti langkah-langkah berikut untuk memasang ReySkripsi di Microsoft Word Desktop (Windows) Anda:

  1. Unduh aplikasi reyskripsi-manager.exe dari tombol di atas. Anda bisa menyimpannya di folder mana saja.
  2. Klik Ganda (Double Click) aplikasi reyskripsi-manager.exe tersebut.
  3. Jendela Installer akan muncul. Klik tombol Install Add-in dan tunggu notifikasi sukses.
  4. Buka aplikasi Microsoft Word, lalu buat dokumen kosong baru.
  5. Pergi ke tab Home (Beranda) > klik menu Add-ins > lalu pilih My Add-ins (Add-in Saya).
  6. Klik ikon ReySkripsi untuk membuka panel di sebelah kanan.
  7. Catatan: Jika sewaktu-waktu muncul pesan notifikasi "Debug Event-based handler" (seperti gambar di bawah), abaikan saja dengan mengklik tombol Cancel.

Jika Anda menggunakan Microsoft Word versi Web (melalui browser), proses pemasangannya sedikit berbeda:

  1. Buka situs Word di Web dan buat dokumen baru.
  2. Pilih tab Home (Beranda) atau Insert (Sisipkan) di bagian atas.
  3. Klik menu Add-ins.
  4. Pada jendela yang muncul, pilih Upload My Add-in (Unggah Add-in Saya).
  5. Pilih file manifest.xml yang bisa Anda temukan di folder source code GitHub, lalu klik Upload.
  6. ReySkripsi kini siap digunakan di panel kanan dokumen Anda.

Jika Anda ingin menghapus add-in ReySkripsi dari komputer Anda, ikuti langkah berikut:

  1. Buka kembali aplikasi reyskripsi-manager.exe yang sudah Anda unduh.
  2. Klik tombol Uninstall.
  3. Tunggu notifikasi sukses, maka Add-in akan otomatis terhapus bersih dari sistem Anda.
  4. Muat ulang (Restart) Microsoft Word jika sedang terbuka.
